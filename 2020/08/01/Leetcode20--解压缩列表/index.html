Leetcode--解压缩列表

Leetcode – 解压缩列表

题目描述：

给一个以行程长度编码压缩的整数列表 nums 。

考虑每对相邻的两个元素 [freq, val] = [nums[2i], nums[2i+1]] （其中 i >= 0 ），每一对都表示解压后子列表中有 freq 个值为 val 的元素，你需要从左到右连接所有子列表以生成解压后的列表。

请返回解压后的列表。

示例：

输入：nums = [1,2,3,4]
输出：[2,4,4,4]
解释：第一对 [1,2] 代表着 2 的出现频次为 1，所以生成数组 [2]。
第二对 [3,4] 代表着 4 的出现频次为 3，所以生成数组 [4,4,4]。
最后将它们串联到一起 [2] + [4,4,4] = [2,4,4,4]。
示例 2：

输入：nums = [1,1,2,3]
输出：[1,3,3]

提示：

2 <= nums.length <= 100
nums.length % 2 == 0
1 <= nums[i] <= 100

解题思路：

创建一个空数组，遍历数组元素下标，当元素下标为奇数时，判断前一位元素是否为0，不为0，就添加奇数位的元素，前一位元素减1直到为0，相当于每逢奇数位就把奇数位的元素添加奇数位前一位元素值这么多次。

代码实现：

class Solution(object):
    def decompressRLElist(self, nums):
        """
        :type nums: List[int]
        :rtype: List[int]
        """
        res = []
        for i in range(len(nums)):
            if i % 2 != 0:
                while nums[i-1] != 0:
                    res.append(nums[i])
                    nums[i-1] -= 1
        return res

复杂度分析：

  • 时间复杂度：O(N2)
  • 空间复杂度：O(1)

其他方法：

（1）遍历数组直接添加元素

算法思路：

以步长为2遍历数组nums，对于当前遍历到的元素a和b，将a个b直接添加到返回数组中即可。

代码实现：

class Solution:
    def decompressRLElist(self, nums: List[int]) -> List[int]:
        ans = list()
        for i in range(0, len(nums), 2):
            ans.extend([nums[i + 1]] * nums[i])
        return ans

复杂度分析：

  • 时间复杂度：$O(N + \Sigma nums_{even})$，其中N是数组nums的长度，$\Sigma nums_{even}$是数组nums中所有下标为偶数的元素之和
  • 空间复杂度：$O(\Sigma nums_{even})$

（2）暴力求解

解题思路：

取一半的nums，然后生成数组。

代码实现：

class Solution:
    def decompressRLElist(self, nums: List[int]) -> List[int]:
        result = []
        for i in range(len(nums)//2):
            result += [nums[2*i+1]] * nums[2*i]
        return result

复杂度分析：

  • 时间复杂度：O(N)，实际上为O(N/2)
  • 空间复杂度：O(偶数索引对应者之和)

（3）分开两列表法

解题思路：

将奇数和偶数位置的数分开成两个列表，[nums::2]是反复次数，nums[1::2]是要出现的数，包装在一起，提取出每个数和其对应的出现次数，显示这个数并且反复相应的次数。

代码实现：

class Solution:
    def decompressRLElist(self, nums: List[int]) -> List[int]:
        return [i for i,j in zip(nums[1::2],nums[::2]) for _ in range(j)]

补充zip函数：

python的zip函数在运算时，会以一个或多个序列作为参数，返回一个元组的列表，同时将这些序列并排的元素配对。

zip()函数操作方法：

  • 使用zip()函数将列表合并，并创建一个元组对的列表。zip()是可迭代对象，使用时必须将其包含在一个list当中，方便一次性显示出所有结果。
a = [1,2,3]
b = [4,5,6]
zip(a,b)
list(zip(a,b))

输出：

[(1, 4), (2, 5), (3, 6)]
  • zip()参数可以接受任何类型的序列，同时也可以有两个以上的参数；当传入参数的长度不同时，zip能自动以最短序列长度为准进行截取，获得元组。
>> l1,l2,l3 = (1,2,3),(4,5,6),(7,8,9)
>>> zip(l1,l2,l3)
[(1, 4, 7), (2, 5, 8), (3, 6, 9)]

>>> str1 = 'abc'
>>> str2 = 'def123'
>>> zip(str1,str2)
[('a', 'd'), ('b', 'e'), ('c', 'f')]

zip函数搭配for循环，支持并行迭代操作。

l1 = [2,3,4]
l2 = [4,5,6]

for (x,y) in zip(l1,l2):
    print x,y,'--',x*y

输出：

2 4 -- 8
3 5 -- 15
4 6 -- 24

（4）两个for循环法

解题思路：

使用列表生成式写出一个源列表，考虑保留那些奇数位置的元素，最后考虑让这些奇数位置的数反复偶数位置的次数。

代码实现：

class Solution:
    def decompressRLElist(self, nums: List[int]) -> List[int]:
        return [nums[i] for i in range(len(nums)) for j in range(nums[i-1]) if i % 2 == 1]

（5）sum方法

解题思路：

获取每个数和它的出现次数，然后将每个出现的数和它的重复次数组成一个列表，将这个列表生成式与一个空列表求和，可以将原二级嵌套列表展开。

代码实现：

class Solution:
    def decompressRLElist(self, nums: List[int]) -> List[int]:
        return sum(([b] * a for a, b in zip(nums[::2], nums[1::2])), [])
