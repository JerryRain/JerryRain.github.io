Leetcode--还原IP地址

Leetcode – 还原IP地址


题目描述：

给定一个只包含数字的字符串，复原它并返回所有可能的 IP 地址格式。

有效的 IP 地址正好由四个整数（每个整数位于 0 到 255 之间组成），整数之间用 ‘.’ 分隔。

示例:

输入: "25525511135"
输出: [“255.255.11.135”, “255.255.111.35”]

解题思路：

使用暴力循环的方法遍历字符串，使用三个指针将字符串拆分为三部分，第一个指针只能指向字符串的前三位，所以第一个指针遍历字符串的前三个元素，第二个指针指向第一个指针的后一位至后四位，同理，第三个指针指向第二个指针的后一位至后四位，从而将字符串分割成四部分，再判断每一个分割好的字符串，如果每个字符串不存在或者大于255或者第一位是0而且字符串长度大于1，都是不合法的IP地址，返回False，否则，满足条件，返回True。

代码实现：

class Solution(object):
    def restoreIpAddresses(self, s):
        """
        :type s: str
        :rtype: List[str]
        """
        n = len(s)
        res = []

        def helper(tmp):
            if not tmp or (tmp[0] == "0" and len(tmp) > 1) or int(tmp) > 255:
                return False
            return True
        
        for i in range(3):
            for j in range(i+1, i+4):
                for k in range(j+1, j+4):
                    if i < n and j < n and k < n:
                        tmp1 = s[:i+1]
                        tmp2 = s[i+1:j+1]
                        tmp3 = s[j+1:k+1]
                        tmp4 = s[k+1:]
                        if all(map(helper, [tmp1, tmp2, tmp3, tmp4])):
                            res.append(tmp1+"."+tmp2+"."+tmp3+"."+tmp4)
        return res

复杂度分析：

  • 时间复杂度：O(n3)

其他解法：

（1）DFS+回溯

思路：

  • 第一步的选择有3种，以“25525511135”为例
    • 选择“2”为第一个片段
    • 选择“25”作为第一个片段
    • 选择“255”作为第一个片段
  • 切第二个、第三个片段都是一样的，面临3种选择
  • 用DFS去遍历所有选择，并且回溯
  • 回溯的原因：某一步切片的选择可能得到一个不合法的地址，得不到正确结果，就不需要往下继续做了，并撤销最后一个选择，回到选择前的状态，去尝试另外一个选择
  • 有一个[选择、探索、撤销选择]的过程

约束条件：

  • 一个片段的长度为1~3
  • 片段的值范围是0~255
  • 不能是"0x"、"0xx"的形式

dfs函数：

  • 选择切出一个长度的片段后，剩余的子串继续递归，可以传子串，也可以穿指向起始位置的指针，加上当前已经生成的片段数组，作为状态，传入dfs函数
  • dfs函数就是[按一定规则，切割从指针start开始的子串]

代码实现：
