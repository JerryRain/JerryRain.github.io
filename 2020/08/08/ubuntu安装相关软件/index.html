Ubuntu系统软件安装

Ubuntu安装相关软件

一、搜狗输入法

1.安装fcitx

sudo apt-get install fcitx-bin
sudo apt-get install fcitx-table

2.打开搜狗输入法官网https://pinyin.sogou.com/linux/，下载32位或者64位

3.安装

打开.deb安装包的下载目录，执行如下安装命令

sudo dpkg -i sogoupinyin_2.2.0.0108_amd64.deb

4.解决依赖问题

sudo apt-get install -f

5.设置搜狗输入法

安装成功后，进入设置

根据红色箭头进入语言安装界面，安装语言（会自动安装中文语言）

根据下方箭头输入框架为fcitx，然后点击上面的Apply System-Wide应用到全局。然后将当前用户进行注销后再进行登录（注销没有效果，重启就可以了）。

登陆后在右上角出现一个键盘标志，点击进入，选择Configure Current Input Method

进入下面的Input Method界面后，选择+号

进入到Add input method界面，将下面的Only Show Current Language 点掉后，在搜索栏搜索搜狗拼音，选中之后进行添加。

添加成功后，将搜狗拼音移到第一位。

成功之后，打开浏览器随便输入，可以看到输入结果，同时成功后下方还会出现搜狗输入法的标志，这时候就可以通过shirt键切换中英文。

当搜狗输入法的拼音出现乱码时，连续按两下shift键可恢复正常。

二、谷歌浏览器

1.下载Google Chrome浏览器

2.安装依赖

sudo apt install libappindicator1 libindicator7

3.进入下载的文件夹，安装谷歌浏览器

sudo dpkg -i google-chrome-stable_current_amd64.deb ＃在deb文件所在目录执行

4.启动谷歌浏览器

/usr/bin/google-chrome-stable

三、微信

1.安装deepin-wine环境

在线安装，通过执行以下命令可以安装最新Release版本的deepin-wine-ubuntu。

wget -qO- https://raw.githubusercontent.com/wszqkzqk/deepin-wine-ubuntu/master/online_install.sh | bash -e

本地安装，通过git clone命令下载到本地，并通过执行install.sh安装。

git clone https://gitee.com/wszqkzqk/deepin-wine-for-ubuntu.git

安装

./install.sh

2.下载微信deepin-wine容器

通过以下命令将微信的deepin-wine容器下载到本地。

wget http://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.com.wechat/deepin.com.wechat_2.6.8.65deepin0_i386.deb

或者直接打开微信下载地址

安装

sudo dpkg -i deepin.com.wechat_2.6.8.65deepin0_i386.deb

然后,使用Win+A搜索WeChat既可打开并登录微信。
