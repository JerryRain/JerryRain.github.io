Leetcode--最大乘积

Leetcode – 最大乘积

题目描述：

给一个整数数组 nums，请选择数组的两个不同下标 i 和 j，使 (nums[i]-1)*(nums[j]-1) 取得最大值。

请计算并返回该式的最大值。

示例 1：

输入：nums = [3,4,5,2]
输出：12
解释：如果选择下标 i=1 和 j=2（下标从 0 开始），则可以获得最大值，(nums[1]-1)(nums[2]-1) = (4-1)(5-1) = 3*4 = 12 。

示例 2：

输入：nums = [1,5,4,5]
输出：16
解释：选择下标 i=1 和 j=3（下标从 0 开始），则可以获得最大值 (5-1)*(5-1) = 16 。

示例 3：

输入：nums = [3,7]
输出：12

提示：

2 <= nums.length <= 500
1 <= nums[i] <= 10^3

解题思路：

双指针，固定一个指针进行第一次遍历，第二个指针从第一个指针的下一位置进行遍历，将两个指针每次遍历的数值分别减1进行相乘，添加进一个新的数组，最后使用max函数返回数组中的最大值。

代码实现：

class Solution(object):
    def maxProduct(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        res = []
        for i in range(len(nums)):
            for j in range(i+1, len(nums)):
                res.append((nums[i]-1)*(nums[j]-1))
        return max(res)

时间复杂度：O(n2)

其他方法：

（1）返回两次最大值进行相乘

代码实现：

class Solution:
    def maxProduct(self, nums: List[int]) -> int:
        first = max(nums)
        nums.remove(first)
        second = max(nums)
        return (first-1)*(second-1)

复杂度分析：

  • 时间复杂度：O(N)
  • 空间复杂度：O(1)

（2）排序法

解题思路：

把列表中的每个元素减1，然后排序，返回最大值和次大值得乘积

代码实现：

class Solution:
    def maxProduct(self, nums: List[int]) -> int:
        a=[nums[i]-1 for i in range(len(nums))]
        a.sort()
        return a[-1]*a[-2]

class Solution:
    def maxProduct(self, nums: List[int]) -> int:
        nums.sort()
        return (nums[-1] - 1) * (nums[-2] - 1)
