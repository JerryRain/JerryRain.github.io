线性代数基础笔记

线性代数基础笔记

一、向量

1.1 什么是向量

向量(vector)：

  • 物理专业学生角度

    向量是空间中的箭头，决定一个向量的是它的长度和它所指的方向，只要长度和方向两个特征相同，就可以自由移动一个向量并保持不变。处在平面的向量是二维的，处在我们所生活的空间向量是三维的。

  • 数学家角度

    向量是有序的数字列表(Vectors ↔ lists of numbers 向量↔数字列表)。

  • 计算机专业学生角度
    向量是有序的数字列表。向量可以是anything，只要保证两个向量相加以及数字与向量相乘是有意义的即可。（向量加法和向量数乘贯穿线性代数始终，二者起着非常重要的作用）。

在线性代数中，一个向量的坐标由一对数构成，可以把它想象成落在x-y二维空间的一个箭头，这对数指导向量起点指向向量终点，第一个数表示沿x轴的长度，正数代表向右移动，负数代表向左移动；第二个数表示沿y轴的长度；如果是三维空间，会有第三个数表示沿z轴运动的长度。

每一个三元数组给出唯一一个向量，每一个向量恰好对应一个三元数组。

为了将向量和点区分开，惯用的方法是将这对数竖着写，然后用方括号括起来，每一对数有且只有唯一对应一个向量。

1.2 向量的基础运算

1.2.1 向量的和

两个向量相加的和，是指两个向量用收尾连接的方法得到的新向量。

向量加法就是把对应项相加，数学公式表达如下：
$$
\left[
\matrix{
a \
b
}
\right]

\left[
\matrix{
c \
d
}
\right]
=
\left[
\matrix{
a+c \
b+d
}
\right]
$$

1.2.2 向量的积
  • 乘以一个大于1的正数 → 表示将一个向量拉伸对应倍数
  • 乘以一个小于1的正数 → 表示将一个向量压缩对应倍数
  • 乘以一个负数 → 表示将一个向量反向延长对应倍数

向量的拉伸或者收缩统称为向量的缩放(Scaling)，向量前的常量称为标量(scalar)，用于缩放向量.

向量与标量的相乘就是将向量中的每个分量与标量相乘，数学公式表示如下：
$$
a *
\left[
\matrix{
x \
y
}
\right]
=
\left[
\matrix{
ax \
ay
}
\right]
$$

两个数乘向量的和被称为两个向量的线性组合。

1.3 向量的线性组合、张成的空间和基

1.3.1 向量的线性组合
  • 一个指向正右方、长度为1，通常被称为“i帽”或者x方向的单位向量
  • 一个指向正上方、长度为1，通常被称为“j帽”或者y方向的单位向量

i帽和j帽这两个向量有着特殊的名称，是xy坐标系的“基(向量)”。(the basis of a coordinate system)

把坐标看成标量时，基向量实际上就是这些标量缩放的对象。

平面内不共线的两个基向量，可以通过任意缩放到达二维平面内的任意一个点，从而构成一个二维平面；如果两个基向量共线，则两个向量组合产生的向量终点只能被限制在一条通过原点的直线上；如果两个基向量都是零向量，则合成的点只能是坐标原点。

1.3.2 张成空间(span)

给定向量的张成空间(span)：所有可以表示为给定向量线性组合的向量的集合。

  • 不共线的两个基向量 → 张成集合为一个二维平面空间
  • 共线的两个基向量（线性相关） → 张成集合是一条直线上的集合

通常就用向量的终点来表示该向量，它的起点仍然位于原点，当我们考虑所有的二维向量时，不必考虑所有箭头，只考虑无限大的二维平面本身。当只考虑一个向量时，把它看做箭头，当考虑多个向量时，就把他们都看做点。

三个向量的线性组合：选择三个标量，对三个向量分别进行缩放，然后把结果相加，这三个向量所有可能的线性组合构成了它们的张成空间。

  • 三个向量不共线 → 张成空间是一个三维立体空间
  • 两个向量不共线，第三个向量与其中一个向量共线 → 张成空间是一个平面
  • 三个向量共线 → 张成平面是一条直线

线性相关 → 有多个向量，并且可以移除其中一个而不减少张成的空间（其中一个向量可以表示其他向量的线性组合，因为这个向量已经落在其他向量张成的空间中）

线性无关 → 如果所有向量都给张成的空间增添了新的维度，它们就是线性无关的

向量空间的一组基是张成该空间的一个线性无关向量集。
