读取rosbag文件画轨迹图

读取rosbag文件画轨迹图


需求：给一个rosbag文件，需要读取文件里面的坐标，并把这些坐标点恢复成轨迹图呈现出来。

方法一

解决方案：使用slam中的一个轨迹查看工具——evo，可用于读取轨迹点坐标的文件，将轨迹复现出来。

实现过程：

1.使用指令rosbag info XXX.bag查看rosbag的具体信息，查看rosbag里面发布的具体topic和消息类型

2.在命令行中通过指令rostopic echo -b XXX.bag -p /topic_name > XXX.csv，将rosbag文件里面的某个主题里面的信息打印到一个csv文件当中（也可以打印到txt文件当中，具体指令为rostopic echo -b XXX.bag -p /topic_name > XXX.txt）

3.下载安装evo工具

sudo apt-get install python-pip  #下载pip工具
pip install evo --upgrade --no-binary evo 或 pip install evo --upgrade --no-binary evo --user  #下载evo工具
pip install numpy --upgrade --user

ps：如果下载完evo工具后直接使用会报错，需要重启一下电脑就好了

4.修改csv文件内容格式，由于evo读取的文件内容有格式要求，所以需要进行修改，内容格式要求如下：时间戳 笛卡尔坐标系三维坐标 四元素坐标，所以一共有8个元素内容，分别为rosbag打印出来的时间戳、x坐标、y坐标、z坐标还有四元素的4个坐标，一共8个元素，每个元素都要用空格隔开，不能用其他字符或者符号隔开，否则evo读取文件内容失败

5.使用指令evo_traj tum XXX.csv --plot即可以读取csv文件并把轨迹显示出来；

6.若想同时显示多条轨迹，可使用以下指令：evo_traj tum XXX.csv --ref=XXX.csv -p --plot -plot_mode xyz即可以显示多条轨迹，xyz为显示的格式，即在三维坐标系下显示，也可以设置为在二维坐标平面下显示轨迹，将xyz参数改为xy、xz、yz即可

7.若想修改显示轨迹线条的颜色、大小等具体参数，可直接在界面内选择参数进行修改

方法二

通过python脚本读取csv文件（csv文件内容格式为只有x轴坐标和y轴坐标，每行的两个坐标值通过空格隔开），将坐标值提取出来，使用matplot将轨迹复现出来

代码实现：

import matplotlib.pyplot as plt
import numpy as np

data_1 = np.loadtxt('/home/jeremy/test.txt')
data_2 = np.loadtxt('/home/jeremy/test1.txt')

plt.plot(data_1[:,0], data_1[:,1],color='red')
plt.plot(data_2[:,0], data_2[:,1],color='green')


plt.legend(('ekf_pose', 'carto_pose'), loc='upper right')  
plt.title('EKF display') 
plt.show()
