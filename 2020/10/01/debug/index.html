Ubuntu开机黑屏

Ubuntu开机黑屏卡在开机界面


问题描述：

由于本机电脑是英特尔集显，不支持安装英伟达显卡驱动，但是糊涂的小编还是安装了，导致开机的时候黑屏，一直一闪一闪（亮晶晶的），而且还会显示[OK]started GNOME Display Manager，但是就是进不去Ubuntu图形化界面。😭😭😭

如下图所示：

问题分析：

由于英伟达驱动和英伟达集显冲突，所以导致开机一直进不去图形化界面。

问题解决：

先删除英伟达驱动，可能删除完英伟达驱动后就可以直接进入图形化界面，但是也可能还进不去，需要删除gdm3图形化软件，重新安装，卸载重新安装后就可以进入图形化登录界面。

问题解决过程记录：

在开机黑屏界面，按Ctrl+Alt+F2（有可能是F1-F6其中一个，有时候还可能是按F6再按F2才能进入命令行模式），进入命令行模式之后，输入用户名和密码进行登录，登录之后按照下面的指令进行操作：

sudo apt-get remove --purge nvidia* #卸载英伟达驱动

sudo apt purge gdm gdm3 #卸载gdm和gdm3

sudo apt-get install gdm3 ubuntu-desktop #重新安装gdm3

sudo systemctl restart gdm #重新启动gdm3

根据上述的指令操作之后，就能够再次进入Ubuntu登录界面继续愉快地使用乌班图系统啦～～～:happy::happy:😂😂😁😁
