Leetcode--好数对数目

Leetcode – 好数对的数目


题目描述：

给一个整数数组 nums 。

如果一组数字 (i,j) 满足 nums[i] == nums[j] 且 i < j ，就可以认为这是一组 好数对 。

返回好数对的数目。

示例 1：

输入：nums = [1,2,3,1,1,3]
输出：4
解释：有 4 组好数对，分别是 (0,3), (0,4), (3,4), (2,5) ，下标从 0 开始
示例 2：

输入：nums = [1,1,1,1]
输出：6
解释：数组中的每组数字都是好数对
示例 3：

输入：nums = [1,2,3]
输出：0

提示：

1 <= nums.length <= 100
1 <= nums[i] <= 100

解题思路：

指定两个指针，选择第一个指针为数组的第一个元素向后遍历，第二个指针为第一个指针选择元素的后一个元素向后开始遍历，如果有相等的元素，就往反馈的结果加1，直到遍历完所有数组元素。

代码实现：

class Solution(object):
    def numIdenticalPairs(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        res = 0
        for i in range(len(nums)-1):
            for j in range(i+1, len(nums)):
                if nums[i] == nums[j]:
                    res += 1
        return res

复杂度分析：

  • 时间复杂度：O(N2)

其他方法：

1.哈希表

循环得到每一个元素在前面出现的次数并放在字典中，返回结果。

class Solution:
    def numIdenticalPairs(self, nums: List[int]) -> int:
        ret, dct = 0, defaultdict(int)
        for i in nums:
            ret, dct[i] = ret+dct[i], dct[i]+1
        return ret

复杂度分析：

  • 时间复杂度：O(N)

2.数学公式法

计算每个数出现的次数n，那么可以构成N * （n - 1）/ 2组，返回它们的总和。

class Solution:
    def numIdenticalPairs(self, nums: List[int]) -> int:
        return sum(v * (v - 1) // 2 for v in Counter(nums).values())

复杂度分析：

  • 时间复杂度：O(N)
