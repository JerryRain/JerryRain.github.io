Leetcode – 左旋转字符串


题目描述：

字符串的左旋转操作是把字符串前面的若干个字符转移到字符串的尾部。请定义一个函数实现字符串左旋转操作的功能。比如，输入字符串"abcdefg"和数字2，该函数将返回左旋转两位得到的结果"cdefgab"。

示例 1：

输入: s = “abcdefg”, k = 2
输出: "cdefgab"
示例 2：

输入: s = “lrloseumgh”, k = 6
输出: “umghlrlose”

限制：

1 <= k < s.length <= 10000

解题思路：

获取原字符串长度l，将字符串分隔为0到n和n到l两段字符串，再将这两个字符串进行拼接。

代码实现：

class Solution(object):
    def reverseLeftWords(self, s, n):
        """
        :type s: str
        :type n: int
        :rtype: str
        """
        l = len(s)
        s1 = s[0:n]
        s2 = s[n:l]
        res = s2 + s1
        return res

class Solution:
    def reverseLeftWords(self, s: str, n: int) -> str:
        return s[n:] + s[:n]

复杂度分析：

  • 时间复杂度：O(N)
  • 空间复杂度：O(N)

其他方法：

（1）遍历列表拼接

算法流程：

  1. 新建一个列表，记为res
  2. 先向res添加”第n+1位至末位的字符“
  3. 再向res添加”首位至第n位的字符“
  4. 将res转化为字符串并返回

代码实现：

class Solution:
    def reverseLeftWords(self, s: str, n: int) -> str:
        res = []
        for i in range(n, len(s)):
            res.append(s[i])
        for i in range(n):
            res.append(s[i])
        return ''.join(res)

复杂度分析：

  • 时间复杂度：O(N)
  • 空间复杂度：O(N)

（2）字符串遍历拼接

算法思路：

遍历n到l区间的每一个元素，将元素逐个添加到返回字符串当中，再遍历从0到n的每一个元素，将每一个元素添加到返回字符串中。

代码实现：

class Solution:
    def reverseLeftWords(self, s: str, n: int) -> str:
        res = ""
        for i in range(n, len(s)):
            res += s[i]
        for i in range(n):
            res += s[i]
        return res

复杂度分析：

  • 时间复杂度：O(N)
  • 空间复杂度：O(N)
