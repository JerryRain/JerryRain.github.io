Leetcode--回文数

Leetcode刷题–回文数


题目描述：

判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

示例 1:

输入: 121
输出: true
示例 2:

输入: -121
输出: false
解释: 从左向右读, 为 -121 。 从右向左读, 为 121- 。因此它不是一个回文数。
示例 3:

输入: 10
输出: false
解释: 从右向左读, 为 01 。因此它不是一个回文数。

思路：

先判断是否为负数，如果是负整数一定不是回文数；如果是正整数，则转化为字符串，再判断正整数的位数，如果是个位正整数，则一定是回文数；如果是偶位数，则回文数的判断标准为左右对称，即左右同时循环遍历各一半如果一样，则是回文数；如果是奇数位数，如果左右指针遍历的元素相同，并且遍历到中间的元素时，左右指针同时指向中间元素，遍历停止，可判断为回文数。（本人太菜，想了一个好麻烦的过程。。。）

代码实现：

class Solution(object):
    def isPalindrome(self, x):
        """
        :type x: int
        :rtype: bool
        """
        s = str(x)
        l = len(s)
        if x < 0 :
            return False
        if l < 2:
            return True
        if l >= 2:
            if l % 2 == 0:
                for i in range(0, l, 1):
                    for j in range(l-1, -1, -1):
                        if s[i] == s[j]:
                            i += 1
                            j -= 1
                            if i == l or j == l:
                                return True
                        else:
                            return False
            else:
                for i in range(l):
                    for j in range(l-1,-1,-1):
                        if s[i] == s[j]:
                            i += 1
                            j -= 1
                            while i == j:
                                return True
                        else:
                            return False

其他解法：

1.同样的思想，但是只用比较前半部分与后半部分即可。

代码实现：

class Solution(object):
    def isPalindrome(self, x):
        """
        :type x: int
        :rtype: bool
        """
        x1 = str(x)
        n = len(x1)
        if n == 1:
            return True
        else:
            x2 = x1[n - n//2:] #截取后半段
            if x1[:n//2] == x2[::-1]: #前半段与反转后的后半段比较
                return True
            else:
                return False

2.切片法–字符串倒转

class Solution(object):
    def isPalindrome(self, x):
        """
        :type x: int
        :rtype: bool
        """
        x = str(x)
        return x == x[::-1]

3.比较输入数字类型

class Solution(object):
    def isPalindrome(self, x):
        """
        :type x: int
        :rtype: bool
        """
        if x < 0:
            return False
        return int(str(x)[::-1]) == x

4.通过取余整除的方法

class Solution(object):
    def isPalindrome(self, x):
        """
        :type x: int
        :rtype: bool
        """
        prev_x = x #定义一个变量存储x的值
        inve_x = 0 #定义一个变量存储x倒序后的值
        while prev_x > 0: #筛选负数，循环过程逆序复现该整数，最终判断是否相等
            inve_x = inve_x * 10 + (prev_x % 10)
            prev_x = prev_x // 10
        return inve_x == x

进阶——蓝桥杯 特殊回文数

问题描述：

123321是一个非常特殊的数，它从左边读和从右边读是一样的。
输入一个正整数n， 编程求所有这样的五位和六位十进制数，满足各位数字之和等于n 。
输入格式
输入一行，包含一个正整数n。
输出格式
按从小到大的顺序输出满足条件的整数，每个整数占一行。
样例输入
52
样例输出
899998
989989
998899
数据规模和约定
1<=n<=54。

解法：

遍历所有五位数和六位数，将数字转成字符串再逆序，然后判断是否为回文数，接着求各位数字之和判断是否等于n，满足以上两个条件即是答案。

n = int(input(''))
for i in range(10000, 1000000):
    num = str(i)
    s = sum(int(j) for j in num)
    if s == n and num == num[::-1]:
        print(num)

缺点：时间复杂度大

改进：

n = int(input(''))
for i in range(10000, 1000000):
    num = str(i)
    if num == num[::-1]:
        if n == sum(int(j) for j in num):
            print(num)
