Leetcode--两数之和

Leetcode刷题–两数之和

题目描述：

给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标。

你可以假设每种输入只会对应一个答案。但是，数组中同一个元素不能使用两遍。

事例：

给定 nums = [2, 7, 11, 15], target = 9

因为 nums[0] + nums[1] = 2 + 7 = 9
所以返回 [0, 1]

我的解法：

每次都确定数组的第一个元素，循环遍历后面每一个数，如果加起来之和为目标值，则停止遍历，返回下标值。

class Solution(object):
    def twoSum(self, nums, target):
        '''
        :type nums: List[int]
        :type target: int
        :rtype: List[int]
        '''
        a = []
        for i in range(len(nums)):
            for j in range(i+1,len(nums)):
                if nums[i]+nums[j] == target:
                    a.append(i)
                    a.append(j)
        return a

缺点：时间复杂度比较高，为O(N2)，空间复杂度为O(1)。

其他解法：

1.求差值，判断差值是否在nums数组里面

class Solution:
    def twoSum(self, nums, target):
        '''
        :type nums: List[int]
        :type target: int
        :rtype: List[int]
        '''
        n = len(nums)
        for x in range(n):
            b = target - nums[x]
            if b in nums:
                y = nums.index(b)
                if y != x:
                    return x,y

时间复杂度：O(n)

2.建立哈希表进行求解

思路：

首先把原来的List建成一个字典，然后新建另一个字典，每次在字典中存储时，dic的key：target - num，value：原List组成的字典值对应的索引。如果检测到某个nums中的值已经存在于新建的字典中，那么证明可以组成一个target值，这时返回此刻遍历到的索引与dict中记录下来的索引。

class Solution:
    def twoSum(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: List[int]
        """
        dic = {}
        for i, num in enumerate(nums):
            if num in dic:
                return [dic[num], i]
            else:
                dic[target - num] = i

时间复杂度：O(N)

3.利用python切片

class Solution:
    def twoSum(self, nums, target):
        for i in range(len(nums)):
            if target-nums[i] in nums[i+1:]:
                return [i,nums.index(target-nums[i],i+1)]

时间复杂度：O(N)

4.双指针

思路：一个指针指向数组第一个元素，另一个指针指向数组最后一个元素，比较这两个数之和与目标值的大小，如果大于目标值则最后一个元素指针左移，如果小于目标值则第一个元素指针右移，如果等于目标值，返回两个元素下标，如果两个指针相遇，说明没有这两个元素，返回空。

代码实现：

class Solution:
    def teoSum(self, nums, target):
        low, high = 0, len(nums)-1
        while low < high:
        if nums[low]+nums[high] == target:
            return low+1, high+1
        elif nums[low]+nums[high] > target:
            high -= 1
        else:
            low += 1

时间复杂度：O(N)，每个元素被遍历一次

空间复杂度：O(1)，只用了双指针，占用常数级空间
