Leetcode--判断子序列

Leetcode – 判断子序列

题目描述：

给定字符串 s 和 t ，判断 s 是否为 t 的子序列。

可以认为 s 和 t 中仅包含英文小写字母。字符串 t 可能会很长（长度 ~= 500,000），而 s 是个短字符串（长度 <=100）。

字符串的一个子序列是原始字符串删除一些（也可以不删除）字符而不改变剩余字符相对位置形成的新字符串。（例如，"ace"是"abcde"的一个子序列，而"aec"不是）。

示例 1:
s = “abc”, t = “ahbgdc”

返回 true.

示例 2:
s = “axc”, t = “ahbgdc”

返回 false.

后续挑战 :

如果有大量输入的 S，称作S1, S2, … , Sk 其中 k >= 10亿，需要依次检查它们是否为 T 的子序列。在这种情况下，应当怎样改变代码？

解题思路：

依次遍历s字符串中的元素，如果s字符串中的元素不在t字符串中，直接返回False；如果在，就使用index函数，使起始指针加1后移一维，再去遍历判断，这样就可以正序遍历s整个字符串元素是否都按顺序出现t字符串当中，如果遍历完都存在，就返回True。

代码实现：

class Solution(object):
    def isSubsequence(self, s, t):
        """
        :type s: str
        :type t: str
        :rtype: bool
        """
        start = 0 
        end = len(t)
        for i in s:
            if i not in t[start:end]:
                return False
            elif start > end:
                return False
            else:
                start = t.index(i,start,end) + 1
        return True

其他解法：

（1）贪心算法

算法思路：

循环s子序列，判断s中的所有字符都会按照顺序出现在t中。可以使用find函数，先遍历s子序列的字符是否在t字符串中，如果在，index加1，find函数就会在index起始位去遍历下一个元素，防止非顺序的情况出现。

代码实现：

class Solution(object):
    def isSubsequence(self, s, t):
        """
        :type s: str
        :type t: str
        :rtype: bool
        """
        index = 0
        for value in s:
            index = t.find(value, index)
            if index == -1:
                return False
            index += 1
        return True

（2）动态遍历

算法思路：

遍历子序列元素，将其提取出来，使用index函数判断是否在t序列当中，有的话指针后移一位，继续调用index函数进行遍历，直到遍历完所有元素，返回True值。

代码实现：

class Solution:
    def isSubsequence(self,s, t):
        j=0#j表示t的当前的index，从0处开始搜索
        for i in range(len(s)):#依据s进行搜索
            temp=s[i]#当前需要搜索的字符为temp
            try:
                res=t.index(temp,j)
            except ValueError:#如果没找到返回错误
                return False
            else:#找到了
                j=res+1#从j的下一个开始搜索
        #for循环结束，说明全部找到了
        return True
