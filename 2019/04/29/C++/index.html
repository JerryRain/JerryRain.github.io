C++学习笔记

C++学习笔记

一.C语言与C++的区别

C语言：面向过程（函数与函数交互）

C++：增加面向对象机制（对象与对象交互）– 一种管理源代码的方式

对C语言的扩充：

1.变量的定义可以出现在程序的任意行

2.提供了标准输入输出流对象

3.用const定义常变量

4.函数重载、函数模板、带默认值的函数

5.引用类型

6.单目作用域运算符

7.string类型字符串

8.使用new和delete代替malloc和free函数等

二.第一个Helloworld程序

  • 将iostream文件的内容添加到程序中；

    1.\n仅仅表示换行
    2.endl表示换行和清空缓存区

1.每条语句占一行

2.每个函数都有一个开始与结束花括号，花括号各占一行

3.函数中的语句对相对于花括号进行缩进

4.与函数名称相关的小括号周围没有空白

5.注释：

  • 单行注释：以”//“开头
  • 多行注释：以/*开头，以*/结尾i

6.编译和执行

  • 编译（预处理->编译->目标文件->汇编）
  • 连接
  • 执行

四.基础知识

第一章：变量、数据类型及运算符

计算机使用内存来记忆或储存计算时所使用的数据。

计算机执行程序时，组成程序的指令和程序所操作的数据都必须存放在某个地方——计算机的内存，也称为主存（main memory）或随机访问储存器（RAM）。

储存单位：bit（位）/ binary digit（二进制数字）

1.变量

定义：变量是计算机中一块特定的内存空间，由一个或多个连续的字节组成。

1 byte 字节 = 8 bit 比特

10M （bit per second）

变量：一个数据储存空间的表示

不同数据存入具有不同内存地址的空间，相互独立

变量的命名：通过变量名可以简单快速地找到在内存中储存的数据

C++语言变量命名规则：

1、变量名（标识符）只能由字母、数字和下划线 3种字符组成

2、名称第一个字符必须为字母或下划线，不能是数字

3、变量名不能包含除_以外的任何特殊字符，如：%、#、逗号、空格等

4、不可以使用保留字/关键字

5、不要使用拼音

6、尽量不要使用单个英文单词

7、提倡使用由正式含义的英文单词

注意：

switch和多重if对比：

相同点：

不同点：

  • switch：等值条件判断——条件是有限个的时候
  • 多重if：判断某个连续区间时的情况

第三章 循环

循环中，可以通过控制循环变量，来控制循环次数。

while循环

基本语法：

while(循环条件)
{
    循环操作语句
}

循环三要素：

  • 循环变量的初值
  • 循环变量的判断
  • 循环变量的更新

特点：先判断，再执行

do-while循环

基本语法：

do
{
    循环操作
}while(循环条件)

特点：先执行，再判断

  • 先执行一遍循环操作
  • 符号条件，循环继续执行
  • 否则循环退出

while和do-while循环的区别：

  • 执行顺序不同
  • 初识情况不满足循环条件时：
    • while循环一次都不会执行
    • do-while循环不管任何情况都至少执行一次

for循环

语法：

for(表达式1;表达式2;表达式3;)
{
    语句;
}

表达式1：通常是为循环变量赋初值，可省略

表达式2：循环条件，是否继续执行循环，可省略

表达式3：更新循环变量的值，可省略

for循环常见问题：

1、忘记定义循环变量或初始化

2、循环条件缺少时会造成死循环

3、循环变量不更新也会造成死循环

4、不可省略分号

break语句

break语句的作用：跳出循环，执行循环之后的语句

continue语句作用：跳出本次循环，继续下次循环

    ......

    ......

{
    ......
    ......

    ......

break和continue对比：

使用场合

  • break可用于switch结构和循环结构中
  • continue只能用于循环结构中

作用（循环结构中）

  • break语句终止某个循环，程序跳转到循环块外的下一条语句
  • continue跳出本次循环，进入下一次循环

循环结构总结：

相同点：

  • 多次重复执行一个或多个任务时考虑使用循环来解决问题

区别：

  • 除了语法不同，判断和执行的顺序也不同
  • 使用情况不同
    • 循环次数确定的情况下，通常选用for循环
    • 循环次数不确定的情况时，通常选用while和do-while循环

第四章.数组及常用算法

1.数组的定义

  • 数组是一个变量，由数据类型相同的一组元素组成。

  • 变量：内存中的一块空间

  • 数组：内存中的一串连续的空间

2.数组的结构和基本要素

  • 标识符：数组的名称，用于去纷纷不同的数组

  • 数组元素：向数组中存放的数据

  • 元素下标：对数组元素进行编号（元素下标都是从0开始）

  • 元素类型：数组元素的数据类型

3.数组的基本信息

  • 数组只有一个名称，即标识符（用来表示数组的变量名）
  • 元素下标标明了元素在数组中的位置，从0开始
  • 数组中的每个元素都可以通过下标来访问
  • 数组长度固定不变，避免数组越界
  • 数组中的所有元素必须属于相同的数据类型

4.数组的使用

语法：datatype arrayName[size]

错误写法：int array[] = {};

初始化数组：

  • 元素个数与声明的一致
  • 若元素未赋值，默认值为0，但至少要给一个元素赋值

C++中的列表初始化新增了一些功能：

  • 省略=号，比如：int days[] {1,15};
  • 大括号可为空，所有元素置零：float m[100] {};
